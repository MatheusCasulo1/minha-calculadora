# Minha Calculadora 🧮

Este é o meu primeiro projeto no GitHub! É uma calculadora simples desenvolvida para praticar e consolidar meus conhecimentos iniciais em desenvolvimento web.

## Tecnologias Utilizadas
* *HTML5*: Estruturação da página.
* *CSS3*: Estilização do layout e dos botões.
* *JavaScript*: Lógica de funcionamento e cálculos da calculadora.

## O que estou aprendendo com este projeto
* Como organizar os arquivos de um projeto web.
* Como criar uma interface de usuário.
* Como fazer o upload de projetos para o GitHub.
